摄像头黑色 解决步聚
1. 检查设备是否被禁用
2. 检查杀毒软件是否关闭
3. 检查摄像头灯是否亮
4. 外接摄像头插口是否插好
5. 检查摄像头是否被其他程序占用
6. 查看QQ是否可用
7. 判断电脑是双显卡还是单显卡
8. 升级驱动
麦克风无声或者声音小
解决步聚
1. 检查麦克风插口是否插好
2. 检查设备是否被禁用
3. 查看自测页面是否音量过小
4. 查看麦克风是否静音或者音量过小
5. 检查系统中麦克风设备是否选择正确
6. 查看QQ麦克风是否正常
7. 升级驱动
扬声器 听不见 解决步聚
1. 确认自测页面音量调节到最大
2. 确认耳机插口是否插好
3. 确认耳机是否有线控开关和线控音量调节键
4. 确认扬声器音量正常，并设置为默认设备
5. 确认杀毒软件是否关闭
6. 确认设备选择正常
7. 查看QQ是否正常工作
8. 升级驱动
